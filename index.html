Iosevka Custom Iosevka is a slender monospace sans-serif typeface inspired by Pragmata Pro, M+ and PF DIN Mono. Fira Code Fira Code is an extension of the Fira Mono font containing a set of ligatures for programming character combinations. Monoid with serifed 'l' Monoid is a semi-condensed, big aperture font designed for programming. This version is customized with a serifed 'l'. Hasklig Hasklig is a fork of the Source Code Pro repository, adding ligatures to make especially Haskell code more readable. Fixedsys Excelsior Fixedsys Excelsior is a 8x16 font with programming ligatures, inspired by DOS and intended for the terminal. DejaVu Sans Code DejaVu Sans Code a is monospaced font with programming ligatures based on DejaVu Sans Mono.
